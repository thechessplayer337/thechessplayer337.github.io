## Projects

### GeoSpy

GeoSpy is a multiplayer GeoGuessr-style social deduction game. Civilians receive screenshots from one country or region, while spies receive screenshots from a similar but different country or region.

- [Play GeoSpy](https://thechessplayer337.github.io/geospy.html)
