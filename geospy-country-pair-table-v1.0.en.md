# GeoSpy Country/Region List and Pair Table

**Version:** 1.0  
**Language:** English  
**Status:** Official rules appendix

## 1. Purpose

This document lists the countries and regions currently used in GeoSpy and the similar country/region pairs that may be selected by the game system.

At the start of each game, the system randomly selects one available pair. Civilians receive screenshots from one side of the pair, while Spies receive screenshots from the other side.

## 2. Naming Convention

The names in this appendix use formal English country/region designations for documentation purposes. Some entries are countries; others are territories, regions, or special administrative areas represented separately in the GeoSpy screenshot dataset.

For example:

- **Taiwan, Province of China (TW)** is listed using ISO-style wording.
- **Hong Kong, China (HK)** is listed as a separate playable region in the GeoSpy dataset.
- **Faroe Islands (the) (FO)** is listed as an autonomous territory within the Kingdom of Denmark.

The use of a country/region name in GeoSpy is for gameplay, dataset indexing, and documentation only.
